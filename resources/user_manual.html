2. Account Code Structure

All account codes follow the format ABC-Xxxx where the first digit after the dash determines the account type. Understanding this structure is essential for correct journal entry posting and financial statement generation.

Account Type Prefixes

PrefixAccount TypeNormal BalanceAppears In
ABC-1xxxAssetsDebitBalance Sheet
ABC-2xxxLiabilitiesCreditBalance Sheet
ABC-3xxxEquityCredit *Balance Sheet
ABC-4xxxRevenueCreditIncome Statement
ABC-5xxxCost of Goods SoldDebitIncome Statement
ABC-6xxxOperating ExpensesDebitIncome Statement
* Equity accounts are Credit-normal except for Drawings and Treasury Stock accounts which are Debit-normal (they reduce equity).

4. Chart of Accounts

Manage the master list of account codes and descriptions used across all journals. Each account has a Normal Balance (Debit or Credit) that drives correct balance calculation in the General Ledger and Financial Statements.

ShortcutAction
Ctrl+NAdd a new account
Ctrl+EEdit the selected account
Ctrl+DDelete the selected account
Ctrl+FFocus the search field
⚠️ Delete All Accounts and Reset to Default buttons are available for resetting the COA. Use with caution — these actions cannot be undone.

11. General Ledger

View all postings for a single account — or all accounts — within a chosen date range, with a running balance column. The running balance respects each account's Normal Balance — credit-normal accounts (ABC-2xxx, ABC-3xxx, ABC-4xxx) show a positive balance when they have a credit balance.

ShortcutAction
Ctrl+FOpen the account selector dropdown
EnterLoad / refresh the ledger
Ctrl+RReset the date range to the current year
ℹ️ Entries are aggregated from all journals (SJ, PJ, CDJ, CRJ, GJ). Selecting All Accounts shows a summary view.

12. Trial Balance

Generate a working trial balance as of any date. Accounts with a positive amount have a debit balance (ABC-1xxx, ABC-5xxx, ABC-6xxx); accounts with a negative amount have a credit balance (ABC-2xxx, ABC-3xxx, ABC-4xxx).

Use the From / To date pickers and click Generate to refresh. Use F5 to reload with the same date range.

13. Financial Statements

Produce the Statement of Financial Position (Balance Sheet) or the Statement of Financial Performance (Income Statement). Enter the company name, click ⚙ Generate Statement…, select the statement type, business type, and date range, then confirm.

ShortcutAction
Ctrl+GOpen the Generate Statement dialog
Ctrl+PPrint the current statement
Ctrl+Shift+PSave the current statement as a PDF
💡 The Business Type selector (Sole Proprietorship / Partnership / Corporation) changes the equity section label and Net Income label on the Balance Sheet automatically.

14. Importing & Exporting Excel Files

Most journal tabs support bulk import and export via Excel (.xlsx). To import, prepare a spreadsheet whose column headers match the expected field names, then press Ctrl+I (or Ctrl+Shift+I in Alphalist). A summary dialog will report the number of records imported and any rows that were skipped.

ℹ️ openpyxl must be installed: pip install openpyxl
